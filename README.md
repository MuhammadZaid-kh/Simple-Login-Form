# Modern Login Form UI

A clean, minimalist, and responsive Login Form built using **HTML5** and **CSS3**. This project focuses on UI/UX fundamentals like spacing, shadows, and flexbox alignment.

## 🚀 Live Demo
You can view the live site here:
`https://muhammadzaid-kh.github.io/Simple-Login-Form/`

## ✨ Features
* **Responsive Design:** Uses Flexbox to ensure the form is centered and looks great on all screens.
* **Modern Styling:** Clean `sans-serif` typography with subtle box-shadows.
* **User Friendly:** Clearly defined input fields and a prominent call-to-action button.
* **Lightweight:** Built with pure HTML and CSS without any external frameworks.

## 🛠️ Built With
* **HTML5:** For the semantic structure of the form.
* **CSS3:** For custom styling, layout, and responsiveness.

## 📂 Project Structure
* `index.html` - Contains both the HTML structure and the CSS styles.
